# *Проект* "Project_2_Jod_Search"

## *Установка*
1. Клонируйте репозиторий:

   https://github.com/Mikhail-156/Project_2._Search_for_vacancies.git
2. Установите зависимости:

   pip install -r requirements.txt 

## *Функционал*
В корне проекта модуль main, запускает работу программы.

В директории src:

- class_hh__API - модуль для взаимодействия с API.
- class_JSON - модуль для взаимодействия с файлами.
- class_vacancies - модуль взаимодействия с вакансиями.
- auxiliary_function - вспомогательные функции.
